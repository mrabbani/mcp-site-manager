Expose WordPress site management as MCP tools for AI clients.

== Description ==
Registers ~73 abilities (posts, pages, media, plugins, themes, options, menus, diagnostics, cache, cron) and exposes them via the MCP Adapter plugin at /wp-json/site-mcp/v1/mcp. Authenticate via Application Passwords.

== Installation ==
1. Install and activate the MCP Adapter plugin first.
2. Activate Site MCP.
3. Visit Settings → Site MCP for the connection URL and an example client config.
